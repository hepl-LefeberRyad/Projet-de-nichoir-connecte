# Projet-de-nichoir-connecte
## Introduction
Dans ce répertoire, nous allons décrire le projet final du premier quadrimestre pour le cours de SmartCities IoT. Le projet consiste en un nichoir intelligent qui prend des photos à l’aide d’un ESP32 équipé d’une caméra intégrée. La caméra est déclenchée par un capteur de mouvement et est accompagnée d’une LED infrarouge pour éclairer l’intérieur du nichoir. Les photos sont ensuite envoyées à un broker via le protocole MQTT et enregistrées dans une base de données. Elles sont également accessibles sur un site web développé à l’aide de Flask.
